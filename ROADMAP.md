# Roadmap

TalkSpark is maintained as a practical open skill for Chinese creator workflows.

## Current Focus

- Keep the core skill readable and portable across AI assistants.
- Improve fact-checking and spoiler-safe rules for hot-topic scripts.
- Expand examples for AI cognition, entertainment commentary, and ad-brief use cases.
- Add lightweight validation so contributors can change references without breaking the bundle.

## Near-Term Ideas

- Add more before/after examples for weak hooks.
- Add a brief-compliance checklist for common Chinese short-video platforms.
- Add a small script linter for banned written-language phrases.
- Add sample workflows for Codex, ChatGPT, Claude, and local skill-folder systems.

## Non-Goals

- Replace human editorial judgment.
- Generate fake personal stories.
- Encourage unsourced claims about real people or companies.
- Become a closed prompt library.
