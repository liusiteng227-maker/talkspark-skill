# Security

TalkSpark is a text-based AI skill bundle. It does not collect data, run a server, or call external APIs by itself.

## Reporting

Please open a GitHub issue if you find a safety or security concern, such as:

- Prompt-injection wording that causes the skill to ignore source checks.
- Guidance that could encourage defamation or unsafe claims about real people or companies.
- Examples that leak private briefs, unpublished campaign material, or personal data.
- A script helper that reads or writes files beyond its documented input.

## Data Handling

Do not include private client briefs, unreleased scripts, personal contact details, or non-public production material in issues or pull requests.

## Fact-Sensitive Content

For public figures, companies, disputes, lawsuits, rankings, recent events, and currently-airing shows, TalkSpark requires source checking or softer wording. See `talkspark/references/fact_check.md`.
