# Examples

These examples show the intended direction. Do not copy them mechanically.

## Rough Idea

```text
很多人不是不会用AI, 是不知道该拿AI干嘛。买了会员, 收藏提示词, 但工作里不用。真正差距是能不能把工作拆成AI能接的小任务。
```

## Better Script

```text
我最近突然意识到一个事, 很多人现在不是不会用AI, 而是不知道该让AI干嘛。这个感觉我自己也特别明显, 电脑里开着ChatGPT, 开着Claude, 旁边还有一个能写代码的AI助手, 明明知道它们都很强, 但只要我当下没有一个具体任务, 我就会盯着输入框发呆。问题不是AI不强, 也不是人笨, 而是我们的工作在脑子里经常还是一团雾。你说帮我提高效率, 这个太大了, AI也不知道从哪下手。但你说帮我把这段会议记录整理成五条待办, 帮我从十个想法里挑三个能拍短视频的选题, 帮我把这段乱稿改得更像真人说话, AI一下就活了。所以我现在越来越觉得, AI时代真正拉开差距的, 不一定是谁会更多提示词, 而是谁能把自己的生活和工作, 拆成一个个可以交给AI协作的小任务。很多时候, 我们不是不会用AI, 我们只是还没学会把自己的工作讲清楚。
```

Why it works:

- Starts from a concrete tool scene.
- Names the real stuck point.
- Gives small task examples.
- Lands on one repeatable sentence.

## Rough Idea

```text
AI时代有优势的人是爱好广泛, 见过世面, 有时间的人。AI给想象力接上手。
```

## Better Script

```text
我最近越来越觉得, AI时代最占便宜的人, 可能不是最懂技术的人, 而是爱好特别杂、见过一点世面、还愿意花时间瞎玩的人。这个话听起来有点不务正业, 但我越想越觉得对。因为AI干了一件特别离谱的事, 它不是替你想象, 它是给你的想象力接上了一双手。以前你脑子里有很多奇怪的想法, 喜欢电影, 喜欢游戏, 喜欢历史, 喜欢商业, 今天想做个网站, 明天想写个故事, 后天又想搞一个小工具, 很容易被说成不专精、想法太多、落不了地。因为你真的落不了地。你想做网页得会前端, 想做视频得会剪辑, 想做游戏得会代码。但现在AI把中间那条很长的技术链拆短了。ChatGPT能帮你整理结构, Claude能帮你扩文本, 一个能写代码的AI助手能帮你搭原型。于是那些看似没用的兴趣, 突然变成了生产资料。你看过的电影, 玩过的游戏, 读过的书, 逛过的城市, 都可能变成提示词里的养料。所以未来最稀缺的, 可能不是会不会操作工具, 而是你每天醒来, 脑子里到底有没有东西想变出来。
```

Why it works:

- Strong identity hook.
- Uses "not X, but Y" reversal.
- Makes the abstract "imagination" visible through tools and actions.
- Ends with a memorable line.
