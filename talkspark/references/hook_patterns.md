# Hook Patterns

Use this reference when the opening line needs to be sharper.

## 1. Audience Lock

Use when the viewer has a clear identity or pain.

Pattern:

```text
如果你也X, 那你可能不是Y, 而是Z。
```

Example:

```text
如果你也买了AI会员, 但一直用不起来, 那你可能不是不会用AI, 而是不知道该让AI干嘛。
```

## 2. Counterintuitive Claim

Use when the idea has identity appeal.

Pattern:

```text
我最近越来越觉得, X时代最占便宜的人, 可能不是A, 而是B。
```

Example:

```text
我最近越来越觉得, AI时代最占便宜的人, 可能不是最懂技术的人, 而是爱好特别杂、见过一点世面、还愿意花时间瞎玩的人。
```

## 3. Concrete Scene

Use when the user provided a lived moment.

Pattern:

```text
昨天/最近/前两天, 我X, 然后突然意识到Y。
```

Example:

```text
前两天晚上, 我开着ChatGPT和Claude, 旁边还有一个能写代码的AI助手, 盯着输入框发呆, 然后突然意识到一个很尴尬的事。
```

## 4. Recognizable Name First

Use for entertainment, show, celebrity, product, or brand scripts.

Pattern:

```text
X这次最有意思的地方, 不是A, 而是B。
```

Example:

```text
王安宇这次在节目里最有意思的地方, 不是他多会唱, 而是他特别像那种会把气氛稳住的人。
```

## 5. Emotional Contradiction

Use when the hook is a feeling the viewer has not named.

Pattern:

```text
有一件事我越想越觉得不对劲, X明明A, 但大家真正卡住的是B。
```

Example:

```text
有一件事我越想越觉得不对劲, AI明明越来越强, 但很多人真正卡住的, 反而是自己不知道要问什么。
```

## Hook Checklist

- Does the first sentence name a person, tool, platform, identity, pain, or conflict?
- Can the target viewer decide within 2 seconds whether to continue?
- Is there a small tension, reversal, or curiosity gap?
- Is it speakable without sounding like a title?
