# Script Structures

Choose the closest structure. Do not label sections in the final script unless the user asks.

## A. Viewpoint Script

Use for abstract ideas, creator observations, and social/AI/career insights.

Flow:

1. One sharp claim.
2. A personal or common scene.
3. Name the hidden problem.
4. Give 2-3 concrete examples.
5. Turn the idea into a memorable sentence.
6. End with a small action or identity recognition.

Good landing:

```text
很多时候, 我们不是不会用AI, 我们只是还没学会把自己的工作讲清楚。
```

## B. AI Cognition Script

Use for AI usage, AI tools, workflow, productivity, and future-of-work scripts.

Flow:

1. Small real usage scene.
2. The mismatch, tool is strong but person is stuck.
3. Reframe, the issue is task design, taste, context, or judgment.
4. Examples using concrete tool names, such as ChatGPT, Claude, Gemini, Kimi, Notion, Cursor, or an AI coding assistant.
5. Practical starting point.
6. Bigger thought, what this changes about work or creativity.

Avoid:

- Empty "AI时代人人都要学习" language.
- Tool lists without lived use.
- Prompt worship without task framing.

## C. Entertainment / Hot Topic Script

Use for drama, variety show, celebrity, and hot topic commentary.

Flow:

1. Name the work/person/event in the first sentence.
2. Mention the visible thing everyone is reacting to.
3. Offer a slightly different angle.
4. Add light background so outsiders can follow.
5. Use one concrete detail from the clip, episode, post, or event.
6. Land on a feeling or judgment, not a thesis.

Default hook:

```text
X这次最有意思的地方, 不是A, 而是B。
```

## D. Ad / Brief Script

Use when a product, show, campaign, or client brief needs a natural creator口播.

Flow:

1. Start with viewing/usage reaction, not the selling point.
2. Mention the recognizable title, person, or use case early.
3. Describe one concrete moment that made the speaker care.
4. Lightly explain why this matters.
5. Close with a soft recommendation or impression.

Avoid:

- "强势来袭"
- "全新升级"
- "给观众带来"
- "不容错过"
- Dense brand benefit lists.

## E. Multi-Account Versions

Use when the user asks for two or more account versions.

Separate by:

- Persona, who is speaking.
- Entry point, what they noticed first.
- Emotional temperature, calmer or sharper.
- Sentence rhythm, compact or flowing.
- Viewer identity, fans, general audience, professionals, parents, creators.

Do not merely replace synonyms.
