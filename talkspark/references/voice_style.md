# Voice Style

Use this reference when a script must feel like a real Chinese creator speaking to camera.

## Core Feeling

The speaker is not teaching from a podium. The speaker is thinking out loud after noticing something, then pulling the viewer into the thought.

Good feeling:

- "我最近突然意识到一个事..."
- "这个感觉我自己也特别明显..."
- "你看, 问题不在AI强不强..."
- "说真的, 这一下就很扎心..."

Bad feeling:

- "在AI高速发展的今天..."
- "我们应该从以下三个方面理解..."
- "这充分说明了AI赋能的重要性..."
- "用户需要建立结构化工作流意识..."

## Spoken Rhythm

Prefer short spoken clauses.

Use commas like breathing points. Use full stops only when a thought should land.

Good:

```text
我最近突然意识到一个事, 很多人不是不会用AI, 而是不知道该让AI干嘛。
```

Bad:

```text
随着人工智能工具能力的不断提升, 普通用户在实际工作场景中的使用阻力已经从技术门槛转移到任务拆解能力。
```

## Personal But Not Fake

Use real details when provided by the user.

If the user says they opened Codex, ChatGPT, and Claude, keep those names.

If the user did not provide a personal scene, do not invent "有一次我..." stories. Use a plausible shared scene:

```text
很多人应该都有过这种感觉, 会员买了, 提示词也收藏了, 但真到工作里还是不知道第一句话该问什么。
```

## Low Teacher Tone

Replace teaching language with discovery language.

- "你必须..." -> "我现在会更倾向于..."
- "正确的方法是..." -> "我发现更好用的方式是..."
- "本质上..." -> "说到底..."
- "这意味着..." -> "所以结果就变成..."
- "首先/其次/最后" -> natural turns like "还有一层", "更麻烦的是", "到这一步就很有意思了"

## Low Marketing Tone

For ads and product briefs, do not list selling points first.

Start from the speaker's actual reaction:

```text
我一开始以为这就是一个普通综艺片段, 结果看着看着发现它最抓人的地方不是热闹, 是那种很具体的情绪。
```

Then lightly connect the product, show, feature, or value.

Do not write:

```text
这档节目聚焦青春、成长、热血与团队协作, 为观众带来全新的视听体验。
```

## Useful Spoken Turns

Use naturally, not all at once.

- 我最近有个特别强烈的感觉
- 这个话听起来有点夸张
- 但我越想越觉得对
- 你发现没有
- 说真的
- 这个很有意思
- 问题就出在这里
- 这一下就很扎心
- 不是说X不重要, 而是Y可能更关键
- 它不是X, 它是Y
- 所以我现在越来越觉得
- 如果你也有这个感觉
